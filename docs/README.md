# Training Studio 训练工作室

一个面向耐力跑者的**个人教练操作台** —— 训练记录、比赛规划、AI 教练对话，集成在同一个网页里。

## 这是什么，不是什么

**它是**：一个数据驱动的、可调教练人格的跑者私人助理。"对话 → 建议 → 落地 → 反馈"形成闭环：把跑步数据、目标赛事、伤病史和长期记忆都喂给 AI，让它当你的第二意见教练，再把它的建议一键落到日历里执行。

**它不是**：又一个跑步打卡 app。如果你只想记录里程，Garmin Connect、佳明、华为运动健康这些手表自带 app 都够用，不需要这个产品。**记录是给教练喂数据的副产品，不是主菜。**

## 给谁用

- **核心用户**：有 A 级目标赛、训练量中高、想用 AI 当陪练或第二意见的跑者。
- **次核心**：想脱离教练自己摸索、但又不想完全裸跑的跑者。

不针对纯小白（不会问对的问题）、精英运动员（已经有人类教练）、纯记录控（手表 app 够用）。

## 能做什么

- **记录训练** —— 手动录入，或上传手表文件批量导入（`.fit` / `.csv`，佳明 / 高驰等）。支持路跑、越野跑、徒步、爬楼、骑行、游泳、力量、HIIT 八类。
- **看趋势** —— 周/月/年跑步距离曲线、路跑强度分布、心率区间时长分布。
- **日历视图** —— 月历同时显示已完成训练和计划训练。
- **管理赛事** —— 给目标比赛打 A/B/C 优先级，记录历史成绩，PR 栏自动展示每类最佳。
- **AI 教练对话** —— 教练实时看到你的资料、近期训练、目标赛事和长期记忆，给个性化建议；还能把建议一键导入到日历。

## 应用结构

四个核心功能区，下文统称"tab"：

1. **Training（训练）** —— 训练记录列表 + 图表子视图。
2. **Calendar（日历）** —— 月历，点某一天加训练或加计划。
3. **Races（赛事）** —— 目标赛事、历史赛事、PR 栏。
4. **AI Coach（AI 教练）** —— 对话、记忆、Prompt 预览。

### 入口位置在两端不一样

| 操作 | 电脑 | 手机 |
|---|---|---|
| 切换 tab | 顶部一排标签 | 底部 5 个图标（多一个 Settings）|
| 个人资料、API Key、语言、退出登录 | 顶部右侧一排按钮 | 底部 Settings tab |
| 修改密码 | 顶部右侧点账号名展开菜单 | Settings tab → Account → Change password |

下文写"打开 X 设置"这种话时，按你当前端找对应入口即可，不再重复列。

## 注册与账号

本应用是**邀请制**。登录页顶部会显示 Training Studio logo，右上角可以先把界面切成中文（中 / EN），选的语言会在注册后带进设置。点「Register / 注册」，填邮箱 + 两次密码 + 一个**邀请码**即可注册。注册后需要先打开邮箱里的验证邮件，完成验证后再回到登录页登录。没有邀请码就找管理员要一个。

登录时可以勾选「在当前设备记住这个账号和密码」。同一台设备上成功登录过的账号，会出现在邮箱栏上方的「已保存账号」下拉里，点选后会自动填入邮箱和密码，方便在几个测试账号之间切换。忘记密码时，登录页可以发送密码重置邮件；如果注册后没收到验证邮件，也可以在登录页重发。登录页底部也有「访问网页版」入口。

新用户填完基础资料后，会自动播放一段简短引导，带你认识每个页面的功能。

**免费体验额度**：新账号不用先申请任何 key，自带 **10 次 AI 教练对话** + **30 次天气刷新**。用完后在设置里填自己的 DeepSeek key / 彩云 token 即可继续；剩余次数在设置里能看到。

需要注销账号时，账号菜单最下方有「删除账号」：要输入两次密码确认，删除会永久清除账号和全部训练数据，无法恢复。
