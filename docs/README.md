# Ultreia

一个面向耐力跑者的**个人教练操作台** —— 训练记录、比赛规划、AI 教练对话，集成在同一个产品里。

## 产品理念

Ultreia 来自圣地亚哥朝圣之路上的一句古老问候。朝圣者在路上相遇时，会用它互相打气，意思接近「继续向前」「向更远处去」。它的传统拼写是 **Ultreïa** —— `i` 上带一个分音符（两个小点），提醒人把它和前面的字母分开、单独读出来。这个来源保留在名字里，提醒这款工具关注的是长线推进，而不是一次性的打卡成绩。

这个名字适合一款耐力训练工具。跑步、越野、铁三、斯巴达或 Hyrox，都不是一次打卡就完成的事。真正重要的是持续记录、复盘、调整，再回到路上。

logo 里那条发光的山路穿过山脊和等高线：山脊代表爬升和长期目标，弯曲路线代表每一次训练选择，背景等高线代表地形、数据和时间层次。它不是一个普通的“健身绿”图标，而是更接近夜间户外地图和耐力训练仪表盘的气质：黑色底座、深苔绿色光感、继续向前。

Ultreia 的意思不是一次给出完美答案，而是每次都往更清楚、更可执行的方向走一点。

开屏问候会从一组纯训练激励语里洗牌轮换；一轮没抽完前不会重复。它不再根据当天计划、训练负荷或晨间状态切换固定句子。

## 这是什么，不是什么

**它是**：一个数据驱动的、可调教练人格的跑者私人助理。"对话 → 建议 → 落地 → 反馈"形成闭环：把跑步数据、目标赛事、伤病史和长期记忆都喂给 AI，让它当你的第二意见教练，再把它的建议一键落到日历里执行。

**它不是**：又一个跑步打卡 app。如果你只想记录里程，Garmin Connect、佳明、华为运动健康这些手表自带 app 都够用，不需要这个产品。**记录是给教练喂数据的副产品，不是主菜。**

## 给谁用

- **核心用户**：有 A 级目标赛、训练量中高、想用 AI 当陪练或第二意见的跑者。
- **次核心**：想脱离教练自己摸索、但又不想完全裸跑的跑者。

不针对纯小白（不会问对的问题）、精英运动员（已经有人类教练）、纯记录控（手表 app 够用）。

## 能做什么

- **记录训练** —— 手动录入，或上传手表文件批量导入（`.fit` / `.csv`，佳明 / 高驰等）。支持路跑、越野跑、徒步、爬楼、骑行、游泳、力量、HIIT 八类。
- **看趋势** —— 周/月/年跑步距离曲线、路跑强度分布、心率区间时长分布。
- **日历视图** —— 月历同时显示已完成训练和计划训练。
- **管理赛事** —— 给目标比赛打 A/B/C 优先级，记录历史成绩，PR 栏自动展示每类最佳。
- **AI 教练对话** —— 教练实时看到你的资料、近期训练、目标赛事和长期记忆，给个性化建议；还能把建议审核成动作卡片后导入日历。

## 应用结构

四个核心功能区，下文统称"tab"：

1. **Training（训练）** —— 训练记录列表 + 图表子视图。
2. **Calendar（日历）** —— 月历，点某一天加训练或加计划。
3. **Races（赛事）** —— 目标赛事、历史赛事、PR 栏。
4. **AI Coach（AI 教练）** —— 对话、记忆、Prompt 预览。

### 入口位置在两端不一样

| 操作 | 电脑 | 手机 |
|---|---|---|
| 切换 tab | 顶部一排标签 | 底部 5 个图标（多一个 Settings）|
| 个人资料、语言、退出登录 | 顶部右侧一排按钮 | 底部 Settings tab |
| 修改密码 | 顶部右侧点账号名展开菜单 | Settings tab → Account → Change password |

下文写"打开 X 设置"这种话时，按你当前端找对应入口即可，不再重复列。

Settings 里还能看到当前版本和更新入口。进入 Settings 时会自动检查一次更新，但同一段时间内会复用上次结果，不会每次切进来都重新请求；也可以手动点 **检查更新** 立即重查。手动检查后如果没有新版本，版本行会提示已是最新版本，按钮会变成查看最近更新说明；收起更新后按钮会回到检查更新。有新版本时安卓 App 会显示下载安装入口。

AI Coach 右上角的收件箱里集中放 **每日推送 / AI 周复盘 / 其他提醒**。每日推送和周报设置也从这里进入；Memory 更新、赛前简报、充值 / 支付提醒等不属于前两类的消息会放在 **其他**。如果 Memory 更新或赛前简报只有教练建议记录、没有系统通知，也可以从其他 tab 回看。周报默认按上周日期周期查看，换设备也能看到账号内已保存的报告。生成过程可以放到后台，完成后安卓会弹系统通知。

启动时会优先保护训练记录：如果网络慢，赛事、AI Coach 对话和每日记录会先显示上次成功加载的本地缓存，并在进入应用后继续后台刷新；训练记录没有可用数据时，才会停在重试页，避免误以为训练被清空。

## 注册与 Aevum 账号

当前产品按个人使用模式运行，不开放自助注册。登录页顶部会显示 Ultreia logo，右上角可以先把界面切成中文（中 / EN），选的语言会在登录后带进设置。

Ultreia 使用同一个 **Aevum 账号**。这个账号也是 Aevum、Viatica 等产品模块共用的账号，不是单独的 Ultreia 账号。

登录时可以勾选「在当前设备记住这个 Aevum 账号和密码」。同一台设备上成功登录过的账号，会出现在邮箱栏上方的「已保存 Aevum 账号」下拉里，点选后会自动填入邮箱和密码，方便在几个测试账号之间切换。忘记密码时，点「Forgot password? / 忘记密码？」会打开重置弹窗并发送密码重置邮件。登录页底部也有「访问网页版」入口。

新用户填完基础资料后，会自动播放一段简短引导，带你认识每个页面的功能。

AI 教练和天气都使用内置服务端配置，不需要自己申请或填写第三方服务密钥。当前个人模式下，AI 和天气请求不检查钱包余额，也不会扣钱包。

需要注销账号时，账号菜单最下方有「删除 Aevum 账号」：要输入两次密码确认。这个操作删除的是整个 Aevum 账号，会删除 Aevum / Ultreia / Viatica 下的所有个人数据，包括训练记录、教练对话、设置和账本数据，无法恢复。删除前可以导出一份 Ultreia 训练数据备份；这份备份不包含 Viatica 账本数据，也不是一键恢复文件。
