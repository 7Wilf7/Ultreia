# 更新日志

> Training Studio 的"装修日志" —— 从 2026-05-13 闲来无事点了下 artifacts 开始，到今天每一天加了什么、改了什么、拆了什么。

## 2026-06-06

- **教练回复流式输出**：用自己的 key 时，教练边想边逐字往外蹦，不用干等一整段。
- **图表新增「近 7 天」**：按天显示最近 7 天的数据。
- **导入审核界面重排**：每行第一行日期 / 类型 / 配速、第二行数据 + 末尾一个小 RPE 框，**导入时就能直接填 RPE**，不用导完再补。
- **训练负荷条挪到训练记录列表顶部**：跟统计、操作按钮一起固定不滑走；填了 RPE / 导入后立即刷新；标注更清楚（近 7 天 · 周均）。
- **海报保存提速**：字体子集化 + 去掉重复编码，存图更快。
- **去掉「睡眠差」当日标记**：已被「今日状态」的睡眠自评取代。
- 应用内使用手册的章节顺序对齐成最新顺序。
- **训练负荷 / 骤增风险**：训练记录列表顶部的训练负荷条，用近 7 天 vs 近 4 周的「时长 × RPE」算 ACWR，颜色提示加得是否太猛；也发给 AI 教练。
- **计划复盘闭环**：过期计划在月历上自动淡出、不用再手动删；点开那天能看到每条计划是完成 / 漏掉 / 跳过，漏的可一键「已完成 / 跳过」。AI 教练会看到最近两周的计划依从，漏了会先问原因。
- **晨间状态自评**：日历里给睡眠 / 腿感 / 精力打个分（差 / 一般 / 好），喂给教练判断今天该顶还是该收。
- **教练更懂计划**：未来约 21 天的计划都会发给教练（不再只发一周、也不再依赖天气），可以直接让它评估计划可行性；导入计划改成**覆盖**当天原计划而不是叠加。
- **周期 / 热适应提醒**：临近目标赛时教练会按堆量 / 巅峰 / 减量阶段说话；比赛日高温高湿会主动给热适应建议。
- **教练消息可复制**：长按教练的回复可以选中、复制文字了。
- **分享海报重做**：换掉原来一排挤在一起、长得差不多的模板，整张海报重新设计成克制的运动杂志风——最大的数字（距离 / PR 数）当绝对主角，配一条主线（单次用真实 GPS 轨迹、周/月/年用「跑量节奏条」、PR 用干净的成绩清单），底部是手写的 Training Studio 签名。选项全部收成下拉框：海报类型、对象（训练 / 时间段 / PR 范围）、风格（日间 / 夜间）、尺寸（竖版 4:5 / 方形 1:1 / 竖屏 9:16）。
