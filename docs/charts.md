# 图表

图表子视图（Training tab → **Charts**）按你选的时间窗口渲染图表。所有图都受 Training tab 顶部 **All activities ▼** 筛选器的影响。

> 默认时间窗口是**最近 4 周**。

## 时间窗口

图表上方几个 preset：

- 最近 **7 天**（按天显示）
- 最近 **4 周** / 最近 **8 周**
- 最近 **6 个月** / 最近 **12 个月**
- 最近 **5 年**

选一个 preset 同时改**所有图表**的窗口。窗口永远以「现在」结尾。

> 这个时间窗口和 Activities 列表的时间标签（All Time / This Week / …）是**独立的**。改图表的窗口不会影响列表的统计范围。

## 1. 主趋势图

最上面这张折线图会**跟着顶部筛选器自动换指标**，统计的是**当前筛选范围内**的训练，桶大小随 preset 是周 / 月 / 年：

| 当前筛选 | 主趋势图显示 |
|---|---|
| 全部 / 路跑 / 越野跑 / 徒步 / 骑行 / 游泳 | 距离（km）|
| 爬楼 | 累计爬升（m）|
| 力量 / HIIT | 时长（分钟）|

- 力量 / HIIT 没有距离，所以在距离视图里不计入。
- 周桶按周一→周日，标签形如 `5-18~24` 或跨月 `5-30~6-5`；月桶按自然月；年桶按自然年。
- 手机端点折线上的某个点，下方会显示那一桶的时间范围和数值。
- 线条下方的填充色纯装饰 —— 「等高线」视觉风格。

> 单独筛**越野跑**或**徒步**时，主图（距离）下面还会多出一张**累计爬升趋势**副图 —— 这两类爬升才是关键看点。

## 2. 路跑类型分布

水平条形图，按当前窗口内 Road Run 的时间分布到 4 个配速子类（Easy / Aerobic / Tempo / Interval）。

**按时长（秒）加权**，不是按次数。理由：一次 90 分钟 tempo 的训练负荷大于 3 次 20 分钟轻松跑，按时长加权更能反映强度分配。

Trail Run / Hiking / Floor Climbing 不进这个图 —— 它们没配速子类。Strength 和 HIIT 也不进。

## 3. 心率区间时长分布

水平条形图，把当前窗口内各次跑步**在每个心率区间（Z1–Z5）停留的时间**加总，按区间展示时长 + 占比，Z1（恢复）冷色到 Z5（最大）热色。

这张图**只统计带逐区间数据的活动**——也就是**用 `.fit` 文件导入的**（手表原始记录里有逐秒心率）。手动录入和 CSV 导入的活动只有平均/最大心率，没有逐区间分布，不进这张图。所以**只有当窗口里有 FIT 导入的跑步时，这张图才出现**。

## 力量训练时长按部位平摊

如果你**用筛选器选中了某个具体部位**（比如「Upper Body」），一次涉及多部位的力量训练会按比例分配时长。例如：一次「30 分钟，上肢 + 下肢」的训练，在「Upper Body」筛选下只贡献 15 分钟，避免把整段 30 分钟全算到一个部位头上。

不筛选具体部位时（看「全部力量」），仍然是完整时长。

## 提示

- 顶部筛选器切了之后图会立刻重渲染，可以用来对比某一类训练在不同时间段的变化。
- 路跑类型条形色从深到浅渐变，呼应强度递增。
- 如果某张图空着，多半是当前窗口里没有对应数据 —— 试着拉长时间窗口（比如从 4 周改到 12 个月）。

## 分享海报

在 Charts 顶部点 **海报**，会打开海报预览。所有选项都是下拉框，从上到下选：

- **海报类型**：单次 / 周 / 月 / 年 / PR。
- **第二个下拉**随类型变化：单次选哪条训练、周/月/年选哪一段时间、PR 选范围（全部 / 今年 / 去年 / 近 12 个月）。
- **风格**：日间（浅色）/ 夜间（深色）。
- **尺寸**：竖版 4:5 / 方形 1:1 / 竖屏 9:16（按你要发的平台挑）。

海报本身是一张克制的运动风排版：最大的那个数字（距离 / PR 数）是绝对主角，配一条主线——单次训练用真实 GPS 轨迹，周/月/年用「跑量节奏条」（每根竖条是一段时间的跑量，最高那根用强调色），PR 用一份干净的成绩清单。底部一行关键数据 + 手写的 Training Studio 签名。

单次海报如果这条训练带 GPS 轨迹，会把轨迹画成主线；没有轨迹则把平均配速放大当主角。周 / 月 / 年只统计实际完成的跑步训练，不含计划。

点 **保存 PNG** 后，Web 版会下载或调起系统分享；Android APP 会直接保存到相册 / Pictures 里的 Training Studio 文件夹，适合发朋友圈、小红书或发给朋友看。
## 移动端查看

手机上折线图会使用更适合小屏的布局，减少坐标刻度和拥挤标签。点折线图上的某个数据点，下方会显示这一桶的时间范围和数值，方便看清具体数据。
