# 产品理念

## 为什么叫 Ultreia

Ultreia 来自圣地亚哥朝圣之路上的一句古老问候。朝圣者在路上相遇时，会用它互相打气，意思接近「继续向前」「向更远处去」。

这个名字适合一款耐力训练工具。跑步、越野、铁三、斯巴达或 Hyrox，都不是一次打卡就完成的事。真正重要的是持续记录、复盘、调整，再回到路上。

Ultreia 想表达的不是「今天必须更快」，而是：

- 你知道自己为什么训练。
- 你能看见过去几周发生了什么。
- 你能把目标赛拆成接下来几天能执行的安排。
- 你有一个可以反复追问、反复校准的第二意见教练。

## 产品不是普通打卡本

Ultreia 的核心不是「记一条训练」。记录只是原材料，真正的价值在后面：

1. 把训练、比赛、身体状态和长期偏好放在一起看。
2. 让 AI 教练基于这些上下文给建议。
3. 把建议落到日历里执行。
4. 训练后再回到记录和复盘，继续修正下一步。

所以它更像一个个人教练操作台，而不是单纯的跑量统计工具。

## logo 的设计思路

logo 里的山形线条来自耐力运动里最常见的两个意象：路线和山脊。它既像一段起伏的山线，也像一次训练周期里的峰谷。

背景的等高线代表地形，也代表训练数据的层次。单次训练看起来只是一个点，连续记录后才会慢慢形成可读的轮廓。

右下角的绿色短线是一个克制的前进标记。它不是奖牌，也不是终点线，更像在地图上给下一段路做的记号：今天到这里，下一步继续。

整体选择深色底和白色主线，是为了让图标在手机桌面、开屏、海报和深色界面里都保持清楚。它不追求像运动品牌那样喊口号，而是更接近训练日志、地图和教练笔记的气质。

## 使用时的判断原则

如果你不知道该从哪里开始，可以按这个顺序用：

1. 先把最近的训练和目标赛补完整。
2. 看图表和日历，确认训练量、强度和恢复有没有明显问题。
3. 把具体问题交给 AI 教练，例如「这周量是不是太大」「周末长距离怎么安排」「赛前两周怎么减量」。
4. 把能执行的建议放进日历。
5. 训练后继续记录，再让下一次建议基于新的现实调整。

Ultreia 的意思不是一次给出完美答案，而是每次都往更清楚、更可执行的方向走一点。
